## 乱序执行



程序中指令不许一条接一条安装程序中的出现顺序执行：

```
#1 x = 2 * c
#2 y = c + 4
#3 z = x + y
#4 a = 4 * c
#5 p = c - 3
```



有时可以通过改变指令的执行顺序提高计算机的速度：#4 和 #5 可以在任何时候执行，但 #3 必须在 #1 和 #2 结束后执行。
